# Portfolio cuts — 18 images

**Paste folder:**
```
public\images\cuts\
```

## File names

| File | Shows as |
|------|----------|
| `g (1).jpg` (or .png, .webp, etc.) | Cut #01 |
| `g (2).*` | Cut #02 |
| … | … |
| `g (18).*` | Cut #18 |

**Note:** There is a **space** between `g` and `(1)` — e.g. `g (1).jpg`  
`g(1).jpg` without the space also works.

Any image format works: jpg, jpeg, png, webp, gif, avif, bmp, svg.
